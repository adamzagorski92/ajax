Ajax - połączenie kilku technologii,
- Zmiana kawałeczków strony,
- skrypt pobierający dane, które uaktualniają dane,
- giełdy mogą korzystać z Ajaxa,
- blogi też mogą 
- można sprawdzić skrypty w Network>preview (F12)

AJAX – Asynchronous Javascript And Xml – to kombinacja użycia technologii, która pozwala pobrać za pomocą JavaScript z serwera dane i umieścić je na stronie, bez ponownego przeładowania tej strony (czyli asynchronicznie).

Technologie składające się na AJAX:

JavaScript (obiekt XMLHttpResponse)
HTML
CSS
XML
JSON
